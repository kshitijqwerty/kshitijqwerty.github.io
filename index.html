The Blue Whale Challenge

Road Detection using Computer Vision
The project aimed at analyzing social profiles which could be the possible victims of the challenge, and if possible detect curators and propagators. We also analyzed the working and statistics of the game. Task was to detect roads using computer vision techniques using color (HSV features) and texture (Haralick features) based features.
